# static

修饰函数的局部变量：在函数的栈退出之后，静态变量的值仍然保存

类的静态变量：在类被加载时，进行初始化；与对象无关

类的静态函数：属于类，而不是对象，无this指针.

类的静态代码块：static {} 在类加载时，执行，只执行了一次，即使创建了多个对象

类的非静态代码块：{} 在类的静态代码块之后，在构造函数之前进行执行．
